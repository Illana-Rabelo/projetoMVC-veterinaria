Bem-vindo

Este sistema implementa 3 CRUDs simples com persistência usando Spring Boot, Thymeleaf e Spring Data JPA (H2).

Tutores

Cadastrar, listar, editar e excluir tutores.

Animais

CRUD de animais com vínculo ao tutor.

Localizadores

CRUD de localizadores.

Operação mestre/detalhe

Exemplo implementado: Venda (mestre) com itens (detalhe), vinculando Cliente e Produto, com baixa/restauração de estoque.

Clientes

CRUD de clientes (cadastro base para vendas).

Produtos

CRUD de produtos com campo de estoque e preço.

Vendas

Criar venda e gerenciar itens (mestre/detalhe).

Relatórios

Consultar indicadores (Top vendidos, estoque baixo, etc.).
