# projetoMVC-veterinaria

Find-Pet: Conectando Tutores e Animais Perdidos

    Sobre o Projeto
O Find-Pet é um sistema idealizado para atender a sociedade civil, especialmente tutores de animais e usuários comprometidos com o bem-estar animal. Nosso objetivo principal é ajudar tutores a localizar seus animais de estimação perdidos.
O sistema funciona realizando buscas a partir de informações detalhadas fornecidas pelo responsável pelo animal, como características físicas (cor, raça, tamanho, idade) e o último local onde o animal foi visto.


    Funcionalidades Principais

Cadastro de Animais Perdidos: Permite que o Tutor publique um anúncio com o máximo de detalhes do pet, incluindo foto obrigatória e localização de onde foi visto pela última vez.

Cadastro de Animais Encontrados: O Usuário Localizador pode publicar uma foto e a geolocalização de um animal que encontrou.

Notificação de Correspondência: O sistema realiza uma "leitura cruzada" das características e notifica o tutor do pet perdido sobre possíveis correspondências.

Gerenciamento de Anúncios: Permite a exclusão de anúncios quando o animal for encontrado, removendo a publicação do feed.
